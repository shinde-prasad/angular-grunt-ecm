Skype search page contains 9 fields that can be used as criteria for your search:
  • Start date: start date and time
  • End date: end date and time
  • Flagging rule: custom flagging rule
  • Custodian: custodians associated with the call
  • Keywords: keywords associated with the call
  • Content: content
  • From: calling party
  • To: called party
  • Duration from (sec) voice duration from
  • Duration to (sec) voice duration to
