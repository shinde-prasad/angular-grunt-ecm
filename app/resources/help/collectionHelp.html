How to merge collections

In order to merge a set of collections you need to select the checkboxes of the collections that you want to merge and click the Merge button.

In the Merge Collection popup select the destination collection by selecting a collection in the Destination collection dropdown, and then confirm the action by pressing the OK button.

How to delete collections

In order to delete a set of collections you need to select the checkboxes of the collections that you want to delete and click the Delete button.
After this, the Delete Collection popup is going to appear with a warning message and the name of the collections that were selected to be deleted. Click the OK button to confirm the action

How to display a collection details

In order to display the collection details you need to click the desired collection in the table. After the click the Collection Details page is going to be displayed.
