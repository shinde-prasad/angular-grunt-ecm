Reuters search page contains 6 fields that can be used as criteria for your search:
  • Start date: message start date and time
  • End date: message end date and time
  • Flagging rule: custom flagging rule
  • Keywords: keywords associated with the messages
  • Custodian: custodians associated with the messages
  • Content: messages content
Reuters RCD channel has these 3 additional search fields:
  • From: chat sender
  • To: chat receiver
  • Transaction Code: transaction unique ID for RCD
After typing and selecting the search criteria you just need to click the Search button located under the form.
