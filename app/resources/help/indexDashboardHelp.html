This page presents you historic data related to the indexing process.
  • Success of last index run: Displays how many documents were successfully indexed and how many failed during the last indexing process
  • Object success history: Displays a trend on how many documents are successfully indexed daily
  • Object total history: Displays a trend on how many documents are sent to the index process daily
  • Object failure history: Displays a trend on how many documents fail to be indexed daily
Clicking the Show more link will display the information broken down by tenant or namespace Clicking the Show less link will display information summed up information
