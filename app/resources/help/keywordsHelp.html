How to create a keywords rule

In order to create a new keyword you need to click the New button.

In the New Keyword popup enter the name and description for the new keyword and then confirm the action by pressing the OK button.

How to delete a set of keywords

In order to delete a set of keywords you need to select the checkboxes of the keywords you want to delete and click the Delete button.
After this, the Delete Keywords popup is going to appear with a warning message and the name of the keywords that were selected to be deleted. Click the OK button to confirm the action

How to display a keywords details

In order to display the keyword details you need to click the desired keyword in the table. After the click the Keywords Details page is going to be displayed.
