Voice search page contains 5 fields that can be used as criteria for your search:
  • Start date: conversation start date and time
  • End date: conversation end date and time
  • Flagging rule: custom flagging rule
  • Duration from: conversation length starting from
  • Duration to: conversation length ending to
After typing and selecting the search criteria you just need to click the Search button located under the form.

Additionally you can click the "Mobile" checkbox and add 3 more filters

  • Order ID: conversation order Id
  • Flagging rule: custom flagging rule
  • Content: conversation content
  • Client: client name involved in the conversation

Also you can click the "Dealer Board" checkbox and add 4 more filters
  • Phone ID: Dealer Board phone ID
  • Custodian: custodians associated with the conversation
  • Trader ID: conversation trader identifier
  • Flagging rule: custom flagging rule
Also you can click the "Desktop Phone" checkbox and add 4 more filters
  • Call Type: the call type can be All, Inbound or Outbound
  • Extension: the phone extension number
  • Custodian: custodians associated with the conversation
  • Flagging rule: custom flagging rule
