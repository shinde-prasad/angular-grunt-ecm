ICE Chat search page contains 9 fields that can be used as criteria for your search:
  • Start date: chat start date and time
  • End date: chat end date and time
  • Flagging rule: custom flagging rule
  • Custodian: custodians associated with the chat
  • Keywords: keywords associated with the chat
  • Content: chat content
  • From: chat sender
  • To: chat receiver
  • Subject: chat subject
After typing and selecting the search criteria you just need to click the Search button located under the form.
